The following wiki should clarify topics from AI - compressed in less information. It is alphabetically sorted. This happens with the intention to build an Anki card deck. Spaced repetition is super efficient to not forget learned things [1].

A

ANN (Artificial neural network) - Processes information through connected units. An analogy is our brain with neurons. It is an umbrella term, there are different properties

by ANNs.

B

Backpropagation - Given an artificial neural network and an error function, the method calculates the gradient of the error function with respect to the neural networks weights.

Bayesian programming - Formalism and methodology for having a technique to specify probabilistic models and solve problems when less than the necessary information is available.

Big O notation - Is an equation how the runtime changes over time with different inputs of variables.

Boolean algebra - In mathematics and mathematical logic, Boolean Algebra is the branch of algebra in which the values of the variables are the truth values true and false, usually denoted 1 and 0.

C

Clustering - Is the task of grouping a set of objects in such a way that objects in the same group are more similar to each other than to those in other groups.

Computer vision - It seeks to understand and automate tasks that the human visual system can do.

CNN (Convolutional neural network) - Is an artificial neural network which is used for image processing and audiodatas.

CPU (central processing unit) - Is the electrical circuits who executes the commands that a computer and the programs on the computer are working.

D

Dartmouth workshop - The Dartmouth summer research project on AI was 1956 a summer workshop which counts as the founding event of AI.

Decision trees - It's a model which supports decisions in the form of a tree model and the possible consequences, including chance event outcomes, resource costs and utility.

decision node = square, chance node = circle, end node = triangle

E

Error function - Is the function which shows how far away the result of a neural network is from the actual aimed result.

F

FLOPS (Floating point operations per second) - Is the amount of floating point operations per second which a computer or everything on what floating point operations is able to perform.

Formal language - The alphabet of a formal language consists of symbols, letters, or tokens that concatenate into strings of the language.

Fuzzy logic - Fuzzy logic is a form of logic in which the truth value of variables may be any real number between 0 and 1. It is employed to handle the concept of partial truth, where the truth value may range between completely true and completely false.

G

Gradient - Holds the partial derivative of a function in one vector.

GPU (Graphics processing unit) - It's an electrical circuit who is performing tasks well iIf the computing tasks are very similar (the equations which are to calculate) for example.

S

SAE Classification - Different levels of the automation of cars:

  • Level 0: Warnings and short intervention from the car.
  • Level 1: Driver and car sharing tasks which are necessary for driving the car but the driver is every moment ready to control every function of the driving the car (cruise control, parking assistance)
  • Level 2: Car takes full responsibility for driving but the driver needs to be able to intervene at every moment - “hands off”.
  • Level 3: There is no duty of being fully focused on traffic while driving so that working on a laptop in the car is possible. Still the driver needs to be ready to intervene - “eyes off” is the motto.
  • Level 4: Driver is not needed. Car is able in specific areas to drive from its own totally autonomous - “mind off”.
  • Level 5: A car which drives at every condition and every geo zone - "steering wheel optional".

Strong AI (AGI, general AI) - AI which is on an intelligence level >= an human and is able to perform a big variety of tasks.

Supervised learning - Infers a function based on the generated outputs of the network from the inputs. With the network and the “integrated” function you can basically start with feeding unknown data.

Every new input in the clustering gets based on the first categorization assigned (think about clustering for a family car where the x axis is horsepower and y axis is price).
