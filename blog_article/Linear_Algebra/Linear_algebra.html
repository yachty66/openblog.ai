Linear Algebra


Linear algebra is a way of dealing efficient with linear equations. The "basic units" - that every \(n\)-dimension has an ordered place - are vectors and matrices.

Vector

\[x=\begin{bmatrix} x_1 \\ x_2 \\ ... \\ x_n \end{bmatrix}\]

\(x\) = Description for the vector

\(x_1 .. x_n\) = \(n\) - Dimensions of the vectors

Matrix

\[x=\begin{bmatrix} a_{11} && a_{12} && ... && a_{1n} \\ x_{21} && a_{22} && ... && a_{2n} \\ ... && ... && ... && ...\\ x_{m1} && a_{m2} && ... && a_{mn} \end{bmatrix}\]

\(x_{11} .. x_{mn}\) = \(mn\) - Dimensions of the matrix

Vector - Vector Products

\[x^Ty \in \mathbb{R} = \begin{bmatrix}x_1 && x_2 && ... && x_n\end{bmatrix} \begin{bmatrix}y_1 \\ y_2 \\ ... \\ y_n\end{bmatrix} = \sum_{i=1}^{n}x_iy_i\]

\(x^T\) = is the transpose of \(x\) (switching of the row and column indices because number of columns has to match the number of rows between the vectors)

\(x^Ty \in \mathbb{R}\) = x and y in each vector is a element of the real numbers \(x\)

\(\sum_{i=1}^{n}x_iy_i\) = forms the sum with the associated \(x\) and \(y\) at the respective places \(i\)

note = this process repeats itself also for matrix - vector and matrix - matrix products

Identity Matrix

\[I_{ij}= \begin{cases} 1 & i = j \\ 0 & i\neq y\\ \end{cases} \]

note = this expressions shows a case distinction with ones on the diagonal and zeros everywhere else. The identity says, that for example two expressions A and B produce the same value for all values of the variables in a certain range.

Diagonal Matrix

\[D_{ij}= \begin{cases} d_i & i = j \\ 0 & i\neq y\\ \end{cases} \]

note = this expressions shows a case distinction where every non-diagonal elements are zeros.
