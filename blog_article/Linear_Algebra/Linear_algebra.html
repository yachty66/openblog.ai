Linear Algebra


Linear algebra is a way of dealing efficient with linear equations. The "basic units" - that every \(n\)-dimension has an ordered place - are vectors and matrices.

Vector

\[x=\begin{bmatrix} x_1 \\ x_2 \\ ... \\ x_n \end{bmatrix}\]

\(x\) = Description for the vector

\(x_1 .. x_n\) = \(n\) - Dimensions of the vectors

Matrix

\[x=\begin{bmatrix} a_{11} && a_{12} && ... && a_{1n} \\ x_{21} && a_{22} && ... && a_{2n} \\ ... && ... && ... && ...\\ x_{m1} && a_{m2} && ... && a_{mn} \end{bmatrix}\]

\(x_{11} .. x_{mn}\) = \(mn\) - Dimensions of the matrix

Vector - Vector Products

\[x^Ty \in \mathbb{R} = \begin{bmatrix}x_1 && x_2 && ... && x_n\end{bmatrix} \begin{bmatrix}y_1 \\ y_2 \\ ... \\ y_n\end{bmatrix} = \sum_{i=1}^{n}x_iy_i\]

\(x^T\) = is the transpose of \(x\) (switching of the row and column indices because number of columns has to match the number of rows between the vectors)

\(x^Ty \in \mathbb{R}\) = x and y in each vector is a element of the real numbers \(x\)

\(\sum_{i=1}^{n}x_iy_i\) = forms the sum with the associated \(x\) and \(y\) at the respective places \(i\)

note = this process repeats itself also for matrix - vector and matrix - matrix products.

Identity Matrix

\[I_{ij}= \begin{cases} 1 & i = j \\ 0 & i\neq y\\ \end{cases} \]

note = this expressions shows a case distinction with ones on the diagonal and zeros everywhere else. The identity says, that for example two expressions A and B produce the same value for all values of the variables in a certain range.

Diagonal Matrix

\[D_{ij}= \begin{cases} d_i & i = j \\ 0 & i\neq y\\ \end{cases} \]

note = this expressions show a case distinction where all non-diagonal elements are zeros.

The Transpose

\[(A^T)_{ij} = A_{ji}\]

note = is the flipping of the rows and columns.

Symmetric Matrices

\[A = A^T\]

note = symmetric matrix.

\[A = -A^T\]

note = anti symmetric matrix.

The Trace

\[trA = \sum_{i=1}^{n}A_{ii}\]

note = is the sum of diagonal elements in the matrix.

Norms

\[\Vert x \Vert_{2} = \sqrt{\sum_{i=1}^{n}x_{i}^2}\]

\(n\) = \(n\) elements of the vector.

note = also known as the "SRSS" operation which is an acronym for square root of the sum of squares. The norm measures the "length" of an vector.

Linear Dependence/Independence

\[x_{n} = \sum_{i=1}^{n-1} \alpha_i x_{1}\]

\(x_n\) = the \(x_n\) vector of an set of vectors \(\{x_1, x_2, ..., x_n\}. \)

\(\alpha_i\) = some scalar \(\{x_1, x_2, ..., x_n\}. \)

note = if the expression in the figure is fullfilled \(x_n\) is linearly independent else linearly dependent.

Rank

column rank = number of linearly independent columns of an matrix \(A\).

row rank = number of linearly independent rows of an matrix \(A\).

rank = column rank of \(A\) is equal to the row rank of \(A\), and so both quantities are called rank \(A\).

The Inverse

\[A^{-1}A = I = AA^{-1}\]

\(A^{-1}\) = inverse of the matrix \(A\).

note = the product of the inverse \(A^{-1}\) and \(A\) is equals to the identity \(I\).

Orthogonal Matrices

\[U^{T}U = I = UU^{T}\]

\(A^{-1}\) = inverse of the matrix \(A\).

note = two vectors are orthogonal if \(x^{T}y = 0\) and normalized if \(\Vert x \Vert_{2}\). In the case for a matrix this has to be the case for all columns to each other to be orthonormal.

Range and Nullspace of a Matrix

\[span(\{x_{1},... x_{n}\}) = \{v:v = \sum_{i=1}^n \alpha_i x_i, \alpha_i \in \mathbb{R} \}\]

\(span(\{x_{1},... x_{n}\})\) = set of all vectors that can be expressed as a linear combination of \(\{x_{1},... x_{n}\}.\)

\[R(A)= \{v \in \mathbb{R}^m: v = Ax, x \in \mathbb{R}^n\}\]

\(R(A)\) = range of a matrix \(A\) is the span of the columns of \(A\).

\[N(A)= \{x \in \mathbb{R}^n: Ax=0\}\]

\(N(a)\) = nullspace of a matrix is the set of all vectros that are equal \(0\) when multiplied by \(A\).

The Determinant

\[S = \{v \in \mathbb{R}^n : v = \sum_{i=1}^n \alpha_i a_i \space where \space 0 \leq \alpha_i \leq 1, i = 1, ..., n\}\]

\(S\) = the span but with coeffients in the range of \(0 \leq \alpha_i \leq 1\). The absolute value of the determinant of \(A\) is a measure of the "volume" of the set \(S\).
